# Loading Calculation Efficiency Workflow

## Step 1: Define Order
1.  Navigate to **Home** > **New Order**.
2.  Select **Global Length** (e.g., 12m, 13m).
3.  Use the table to add pipe types:
    - Search for diameter (e.g., "250").
    - Select pressure class (PN10, PN16).
    - Enter quantity.
4.  *Optional*: Upload CSV directly (`DN, PN, Qty`).

## Step 2: Optimization
1.  Click **Calculate Load** button.
2.  The system processes the order (~5-10 seconds).
    - **Phase 1**: Bundling (Nesting).
    - **Phase 2**: Truck Packing.

## Step 3: Analysis (Dashboard)
1.  Review **Summary Cards**:
    - Total Weight vs Payload Capacity.
    - Number of Trucks.
2.  Inspect **Truck 1**:
    - View 3D visualization.
    - Check "Loading Efficiency" (e.g., 98%).
    - Note instructions: "Bundle TPE400 contains TPE250".

## Step 4: Export
1.  Click **Export PDF**.
2.  Download the "Loading Ticket".
3.  Send to:
    - Warehouse Manager (for assembly).
    - Driver (for transport documentation).
